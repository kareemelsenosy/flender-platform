Importing Sheets

ℹ Each Google Sheets URL imports as its own result, and different sheets run in parallel.

How it works

  1. Add one or more Google Sheets URLs above
  2. Select which tabs/sheets to import (all tabs selected by default)
  3. Toggle "Save images" to include images from =IMAGE() formulas and Dropbox links
  4. Each added Google Sheet imports simultaneously as its own result
  5. Selected tabs within the same sheet stay together in that sheet's import
  6. Review and export each imported sheet separately, then search for missing images if needed
